## Quick start (Python CLI)

1. Create a virtual environment (optional but recommended):

   python -m venv .venv
   source .venv/bin/activate   # on Windows use `.venv\Scripts\activate`

2. Install dependencies:

   pip install -r requirements.txt

3. Run the example:

   python hello.py

4. Run tests:

   pytest
